版本更新单使用文档
=========================== 

[TOC]

为了规范游戏项目的运维规范，游戏的版本更新需要走审批流程。通常由运营发起版本更新工单，并且指定更新项目的各个环节审批人来完成审批。审批完成后，运维部相关人员会收到更新请求，然后执行更新单上的更新。

**在前后端的负责人审批过程中，需要填写更新版本号以及更新的注意事项**

在某些情况下，如果遇到审批人请假或者不方便审批，工单发起人可以在**我的申请**中修改该节点的审批人
